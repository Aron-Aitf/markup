# Fish

| Keybinding                      | Effect                                        |
| ------------------------------- | --------------------------------------------- |
| `Ctrl + K`                      | Delete from the cursor to the end of the line |
| `Ctrl + C`                      | Delete current command                        |
| `Ctrl + Left Arrow/Right Arrow` | Move cursor one word to the left or right     |
| `Ctrl + R`                      | FZF fuzzy find history                        |
| `Alt + Right Arrow`             | Accept first word of suggestion               |
| `Alt + C`                       | FZF fuzzy find files in current directory     |

# Global

| Keybinding   | Effect                  |
| ------------ | ----------------------- |
| `Ctrl + F11` | Open drop-down terminal |
